# WhareScore Badge icons

PNG assets required for Chrome Web Store submission: `icon-16.png`,
`icon-48.png`, `icon-128.png`. These will be generated from the WhareScore
brand SVG at submission time — not tracked in git as binary placeholders.

At build time, if these files are missing, Chrome still loads the extension
(the icon falls back to the default puzzle piece). Generate the PNGs once
the design team signs off on the colour variant.
